Informations Importantes

  • Ce formulaire utilise un modèle de machine learning pour évaluer le risque de maladie cardiaque.
  • Tous les champs sont requis pour une prédiction précise.
  • Les valeurs doivent être basées sur des examens médicaux récents.
  • Les résultats sont à titre informatif uniquement et ne remplacent pas un avis médical professionnel.
  • Consultez toujours un professionnel de santé pour un diagnostic médical.
  • Vos données sont traitées de manière sécurisée et ne sont pas stockées.

Confidentialité

Vos données médicales sont traitées avec la plus stricte confidentialité. Aucune information personnelle n'est requise ni stockée.
